# Chandra Silks Frontend

Frontend for Chandra Silks eCommerce application.

## Tech Stack

* React.js
* Material UI (MUI)
* Axios
* React Router DOM

## Features

* User Authentication
* Product Listing
* Search & Sort Products
* Cart Management
* Order Management
* Responsive Design
* Admin Dashboard

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Project Structure

```text
src/
 ├── components/
 ├── pages/
 ├── protectedRoutes/
 ├── utils/
 └── assets/
```
